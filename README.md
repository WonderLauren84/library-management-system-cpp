# Library Management System (C++)

## Overview
This is a console-based Library Management System created for COS 109 at Kentucky State University.

The program allows users to:
- Search for books
- Check out books
- Return books
- Display member information
- Add books to the library

## Concepts Demonstrated
- Object-oriented programming
- Inheritance
- Composition
- Virtual functions
- Exception handling
- Searching and sorting
- Vectors
- Linked lists
- Stacks and queues
- Templates
- Recursion
